## How to use:
1. [Visit this link](https://discord.com/oauth2/authorize?client_id=1468033247173677086) while signed into Discord and add the bot to your user.
2. To translate text to wingdings, run the `/to-wingdings` command in any channel. Specify the message to translate, and which dialect to use.
3. To translate text from wingdings, right-click a message you want to translate and go to `Apps` -> `Translate Wingdings`.
